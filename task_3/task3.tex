\section*{Задача 3. Три-одиозные числа}
Рассмотрим последовательность чисел трибоначчи: $T_0 = T_1 = 0$, $T_2 = 1$, 
$T_n = T_{n-1} + T_{n-2} + T_{n-3}$ для $n \ge 3$. 
Любое неотрицательное целое число можно записать в трибоначчиевой системе счисления 
используя только цифры $0$ и $1$ без трёх подряд идущих единиц.

Неотрицательное целое число назовём \textit{три-одиозным}, если в его записи 
в трибоначчиевой системе количество единиц нечётное.

\textbf{Формат ввода:}\\
В первой строке число $N$ ($1 \le N \le 1000$). 
В следующих $N$ строках содержатся числа $A_i$ ($0 \le A_i \le 2^{24}-1$).

\textbf{Формат вывода:}\\
Выведите количество три-одиозных чисел среди введённых.

\begin{examplebox}
  \begin{minipage}[t]{0.48\linewidth}
    \textbf{Ввод:}\\
    \begin{verbatim}
4
0
1
2
733
    \end{verbatim}
  \end{minipage}%
  \hfill%
  \begin{minipage}[t]{0.48\linewidth}
    \textbf{Вывод:}\\
    \begin{verbatim}
2
    \end{verbatim}
  \end{minipage}
\end{examplebox}
