\section*{Задача 1. Тетра-злые числа (Лёгкая)}
\textbf{Условие.} Рассмотрим последовательность чисел тетраначчи: $T_0 = T_1 = T_2 = 0$, $T_3 = 1$, а для любого натурального $n \ge 4$: $T_n = T_{n-1} + T_{n-2} + T_{n-3} + T_{n-4}$. Любое неотрицательное целое число $N$ можно единственным образом записать в тетраначчиевой системе счисления в виде $N = d_k T_{k+4} + d_{k-1} T_{k+3} + \dots + d_0 T_4$, где каждая цифра $d_i \in \{0, 1\}$ и среди любых четырёх идущих подряд цифр обязательно есть хотя бы один ноль (запрещено сочетание $1111$).

Неотрицательное целое число назовём \textit{тетра-злым}, если в его записи в тетраначчиевой системе количество единиц \textbf{чётное}. Составьте программу, которая по заданной последовательности десятичных чисел подсчитывает количество тетра-злых чисел в ней.

\textbf{Формат ввода.} В первой строке содержится целое число $N$ ($1 \le N \le 1000$) -- количество чисел. В следующих $N$ строках записаны неотрицательные целые числа $A_1, A_2, \dots, A_N$ ($0 \le A_i \le 2^{24}-1$), каждое на отдельной строке.

\textbf{Формат вывода.} Выведите единственное целое число -- количество тетра-злых чисел во входной последовательности.

\begin{examplebox}
  \begin{minipage}[t]{0.48\linewidth}
    \textbf{Ввод:}\\
    \begin{verbatim}
4
1
0
3
732
    \end{verbatim}
  \end{minipage}%
  \hfill%
  \begin{minipage}[t]{0.48\linewidth}
    \textbf{Вывод:}\\
    \begin{verbatim}
3
    \end{verbatim}
  \end{minipage}
\end{examplebox}
